# ⚡ Calculadora Elétrica

Projeto web simples desenvolvido em **JavaScript, HTML e CSS** com o objetivo de auxiliar em cálculos básicos de eletroeletrônica.

## 📌 Funcionalidades

O sistema possui as seguintes ferramentas:

* 🔋 **Consumo Energético**

  * Calcula consumo diário e mensal
  * Estima o custo com base no valor do kWh

* ⚡ **Lei de Ohm**

  * Calcula tensão (V), corrente (A) ou resistência (Ω)
  * Basta informar dois valores

* 🔄 **Conversão de Unidades**

  * Conversão entre múltiplos e submúltiplos (T, G, M, k, m, μ, n, p)

* 🎨 **Cálculo de Resistores**

  * Determina o valor da resistência a partir das cores
  * Suporte para resistores de 4 ou 5 faixas

* 🔗 **Resistência Equivalente**

  * Calcula resistência em série e paralelo

## 🖥️ Interface

Interface simples com botões interativos que executam cada cálculo através de prompts no navegador.

## 🚀 Como Executar

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Clique em uma das opções para utilizar as funções
